# Hello World
## A Starter Addon Template

*Hello World* is a bare-bones example package designed to teach new developers how to hook into the Fenestra SDK. It demonstrates command registration, basic UI layout, and system chat integration.

### Usage
Type `/hw` in the game chat to trigger the addon.

### Developer Concepts Covered
1. **Command Routing:** How to register a `/` command using `fw.command.register`.
2. **Context Bounds:** How to spawn a basic `window_state` and pin it to the screen.
3. **Colors:** Utilizing the `fw.ui.color` CSS-compliant color tables.

### Notes
Feel free to duplicate this folder and rename the `manifest.xml` and `.lua` file to begin creating your own custom addons.
