# Addon Manager

Welcome to the **Addon Manager** for Fenestra Voli Edition. 

This is your native, lightning-fast dashboard for controlling everything running in the background of FFXI. No more typing out long load commands or digging through text files—just point, click, and play.

## Core Features

* **Instant Hot-Swapping:** Toggle your addons `Active` or `Offline` in real-time. The engine will instantly boot them up or shut them down without needing to reload the game.
* **Smart Profiles:** The Addon Manager watches the network. The exact second you log in, it detects your character and automatically boots up your saved addon loadout for that specific profile. 
* **Safe Core Systems:** Libraries and Core Systems are safely tucked away in their own menu so they don't clutter your screen, but you can still check their versions and read their manuals.

## Chat Commands

Type these directly into your game chat:

* `/addon` - Opens or closes the main dashboard.
* `/addon rescan` - If you manually drop a new addon folder into your hard drive while playing, run this to refresh the list.
* `/addon profile <name>` - Bypasses the UI to instantly switch your active loadout.

### Under the Hood

The Addon Manager directly hooks into the Windower 5 C++ Engine to guarantee crash-proof, native command execution.
